旧发行版上安装新版本Python

  • 2023-06-29
  • 作者 nop
  • 877 字

本文最后编辑于 前，其中的内容可能需要更新。

由于各种原因我们需要在一些旧发行版中安装新版本的Python，通常网上的教程都是进行编译安装，但新手进行编译安装极容易出现依赖缺失等问题，最简单的方法还是使用编译好的包直接进行安装，这里分别介绍Ubuntu和Centos的安装方法。

Ubuntu

1. 添加源

2. 安装

deadsnakes的源在境外且没有国内镜像源可用，下载会很慢。这个时候可以可以访问deadsnakes源地址，提前下载deb包到/var/cache/apt/archives中，再安装就可以使用下载好的缓存。

3. 安装pip

4. 注意

网上有的教程会让替换系统默认Python，这里不建议替换默认版本，这可能导致系统依赖Python的组件（如apt）出现异常，使用新安装的Python使用全名Python3.10即可。

Centos

1. 添加源

2. 安装

同apt一样，dnf也有缓存文件夹/var/cache/dnf，在raven-release源下载rpm包，然后将rpm包放到缓存文件夹下raven-modular-*/packages中即可。

3. pip

该源安装的Python已安装pip，使用python3.10 -m pip即可。
